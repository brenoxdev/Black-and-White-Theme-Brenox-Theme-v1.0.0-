Tema minimalista e focado em performance para Discord, baseado no ClearVision V7.

---

## Funcionalidades

- Ícones de servidores em preto e branco (cor ao passar o mouse)
- Lista de amigos e atividades em escala de cinza
- Interface limpa e minimalista
- Estrutura pronta para efeito glass
- Cabeçalho personalizado

---

## Requisitos

- Discord Desktop
- Vencord instalado

---

## Instalação

### 1. Ativar Transparência (Opcional – Para efeito Glass)

Configurações → Vencord → Ativar Transparência

Se não funcionar, inicie o Discord com:

```
--enable-transparent-visuals
```

---

### 2. Instalar o Tema

1. Baixe `Brenox-Theme.theme.css`
2. Abra Discord → Configurações → Temas
3. Clique em **Abrir Pasta de Temas**
4. Cole o arquivo dentro da pasta
5. Ative o tema
